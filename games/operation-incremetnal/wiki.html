Features

Add

Function: Adds counters by upg2b (counter = counter.add(upg2b))

Multiply

Function: Multiplies counters by upg8b (counter = counter.mul(upg8b))

Power

Function: Raises (exponent) counters by 1.001 (counter = counter.pow(1.001))

Trivia: if you try to use the power button with negative counters, ??? prevents you from using it and talks about i and breaks the 4th wall

Prestige

Function: Resets all progress below prestige to add prestige counters by the amount of counters divided by 10

(counter = zero; prestigecounter = prestigecounter.add(counter.divide(10)))

Counter Generators

Function: The 1st generator generates counters, the 2nd generates 1st generators, the 3rd generates 2nds, and so on. There are 9 generators in total

( counter = counter.add((cgen0.divide(tps).mul(upg2b)).add(cgen1.divide(tps).mul(cgen1multi/2))); cgen1 = cgen1.add((cgen2.divide(tps).divide(20)).mul(cgen2multi)); cgen2 = cgen2.add(cgen3.divide(tps).divide(20).mul(cgen3multi)); cgen3 = cgen3.add(cgen4.divide(tps).divide(20).mul(cgen4multi)); cgen4 = cgen4.add(cgen5.divide(tps).divide(20).mul(cgen5multi)); cgen5 = cgen5.add(cgen6.divide(tps).divide(20).mul(cgen6multi)); cgen6 = cgen6.add(cgen7.divide(tps).divide(20).mul(cgen7multi)); cgen7 = cgen7.add(cgen8.divide(tps).divide(20).mul(cgen8multi)); cgen8 = cgen8.add(cgen9.divide(tps).divide(20).mul(cgen9multi)); )
